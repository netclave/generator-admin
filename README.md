# NetClave Generator Admin

**Turn your local network into a hardened enclave fortress**

## Intro

The main purpose of this component is to manage and configure the Generator. It is implemented in Flutter and is using Bluetooth for communication to the Generator. It also scans QR codes provided by the Web and Mobile wallets. This way, it attaches the wallet to a given Generator component.
